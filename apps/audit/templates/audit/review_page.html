怎么用这个页面

1

先看最近结果

先确认最近生成了哪些归因报告，哪些验证批次需要复盘。

2

进入专题页面

按问题类型进入报告列表、指标表现或阈值验证，不需要手工拼 API 地址。

3

查看操作日志

管理员可直接追踪 MCP/SDK 调用链，普通用户查看自己的审计日志。

4

需要集成再调 API

接口保留给外部系统、脚本和自动化任务，本页只把它们作为辅助入口。
