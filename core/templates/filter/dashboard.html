HP 滤波 (Hodrick-Prescott)

  • 原理: 将时间序列分解为趋势分量和周期分量
  • 参数: Lambda = 129600 (月度数据标准)
  • 特点: 扩张窗口模式，每个时点只用历史数据
  • 适用: 宏观经济趋势分析、周期识别
{% else %}

Kalman 滤波 (局部线性趋势)

  • 原理: 状态空间模型，递归估计水平值和斜率
  • 特点: 单向滤波，支持增量更新
  • 输出: 趋势 + 斜率（变化率）
  • 适用: 实时数据更新、动态趋势跟踪
{% endif %}
