事后归因审计用于分析交易决策的结果，总结经验教训，改进投资策略。

什么是事后归因审计？

事后归因审计是对投资决策的回溯分析，旨在：

  • 理解盈利/亏损的根本原因
  • 验证投资逻辑的有效性
  • 识别决策中的偏差
  • 积累可复用的经验

生成归因报告

为回测结果生成详细的归因分析报告，包括收益分解、损失分析和经验总结。

API 端点：

POST /audit/reports/generate/

回测报告

使用说明

1. 生成归因报告

在回测完成后，调用 POST /audit/reports/generate/ API，传入 backtest_id 即可生成归因报告。

2. 查询报告

使用 GET /api/audit/summary/?backtest_id=1 查询特定回测的归因报告，或使用日期范围查询多份报告。

3. 报告内容

归因报告包含：

  • 收益分解：各策略/因子对总收益的贡献
  • 损失分析：主要亏损交易的归因分析
  • 经验总结：可复用的成功/失败经验
