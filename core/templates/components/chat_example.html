集成代码示例

HTML模板

<!-- 引入样式和脚本 --> <link rel="stylesheet" href="/static/css/components/chat-widget.css"> <script src="/static/js/components/chat-widget.js"></script> <!-- 创建容器 --> <div id="my-chat-widget"></div>

JavaScript初始化

// 基础用法 const chat = new AgomChatWidget({ containerId: 'my-chat-widget', title: 'AI 助手', height: '500px' }); // 带配置的用法 const chat = new AgomChatWidget({ containerId: 'my-chat-widget', title: 'Regime判定助手', placeholder: '请输入关于宏观经济环境的问题...', defaultProvider: 'openai', defaultModel: 'gpt-4', height: '450px', showHeader: true, showModelSelector: true, showProviderSelector: true, enableHistory: true, maxHistoryLength: 50, onMessageSent: (data) => { console.log('发送消息:', data); }, onMessageReceived: (data) => { console.log('收到回复:', data); }, onError: (error) => { console.error('出错:', error); } });

API方法

// 清空聊天历史 chat.clearHistory(); // 设置系统提示词 chat.setSystemPrompt('你是一个专业的宏观经济分析师...'); // 销毁组件 chat.destroy(); // 获取/设置状态 chat.state.currentProvider = 'deepseek'; chat.state.currentModel = 'deepseek-chat';

配置参数说明

显示配置

  • title - 窗口标题
  • height - 高度（如: '500px'）
  • width - 宽度（如: '100%'）
  • showHeader - 是否显示头部
  • showModelSelector - 显示模型选择器
  • showProviderSelector - 显示提供商选择器

功能配置

  • placeholder - 输入框占位符
  • defaultProvider - 默认提供商
  • defaultModel - 默认模型
  • enableHistory - 启用历史记录
  • maxHistoryLength - 最大历史条数
  • apiBaseUrl - API基础路径

回调函数

  • onMessageSent - 消息发送时
  • onMessageReceived - 收到回复时
  • onError - 发生错误时
