数据源配置说明

Tushare Pro

推荐

Tushare Pro 是一个专业的财经数据接口平台，提供股票、基金、期货、宏观经济等多种数据。

获取 API Token 步骤：

  1. 访问 https://tushare.pro/register 注册账号
  2. 登录后点击右上角头像 → 个人中心
  3. 在"用户信息"页面找到"接口TOKEN"
  4. 点击"复制TOKEN"按钮

配置参数：

参数 值 说明
数据源名称 Tushare Pro (主) 自定义名称，建议标注主/备
数据源类型 Tushare Pro 下拉选择
是否启用 ✓ 是 勾选启用
优先级 1 数字越小越优先，主源设为 1
API Key 你的 Token 粘贴从 Tushare 复制的 Token
自定义 HTTP URL 可留空 接入第三方 Tushare 代理源时填写，系统会写入 `pro._DataApi__http_url`

可用数据指标：

SHIBOR - 银行间拆借利率 指数日线数据

AKShare

免费

AKShare 是基于 Python 的财经数据接口库，开源免费，无需注册即可使用。

配置参数：

参数 值 说明
数据源名称 AKShare (备) 作为备用数据源
数据源类型 AKShare 下拉选择
是否启用 ✓ 是 勾选启用
优先级 2 作为备用源，优先级设为 2
API Key 留空 AKShare 不需要 API Key

可用数据指标：

PMI - 采购经理指数 CPI - 居民消费价格指数 PPI - 工业生产者出厂价格指数 M2 - 广义货币供应量

QMT (XtQuant)

本地终端

QMT 行情通道适合把本地 XtQuant / QMT 终端纳入统一财经数据源中台。当前版本只接入行情、技术快照和日线历史 K 线，不包含下单交易。

接入前提：

  1. Windows 环境已安装并可正常启动 QMT / XtQuant
  2. Python 运行环境已安装 `xtquant`
  3. 本机能正常读取 QMT 行情数据

配置参数：

参数 值 说明
数据源名称 QMT 行情 建议区分本地机房或终端实例
数据源类型 QMT (XtQuant) 下拉选择
优先级 15 建议放在东方财富之后、AKShare/Tushare 之前或由你自定
API Key / Secret 通常留空 QMT 行情通道不依赖远程 Token
额外配置 JSON {"client_path":"C:/qmt","data_dir":"D:/qmt/data"} 可选，适合指定本地终端路径、数据目录、复权参数等

当前支持能力：

实时行情快照 技术快照 日线历史 K 线

常见问题 FAQ

Q: 为什么要配置多个数据源？
A: 系统支持主备数据源自动切换。当主数据源失败时，会自动尝试备用数据源，确保数据获取的稳定性。同时可以通过多个数据源进行数据一致性校验。
Q: 优先级数字越小越优先吗？
A: 是的。系统会按照优先级从低到高的顺序尝试数据源。建议将 Tushare Pro 设为优先级 1（主），AKShare 设为优先级 2（备）。
Q: 如何知道数据源是否正常工作？
A: 可以在"宏观数据管理"页面查看数据同步情况。如果某个数据源一直失败，系统会自动切换到备用源。建议定期检查 Admin 后台的数据记录。
Q: Tushare Pro 有积分限制怎么办？
A: Tushare Pro 采用积分制，普通用户每日有一定的调用次数限制。建议将 AKShare 配置为备用源，可以分担部分请求。如需更高额度，可以考虑升级会员。
Q: 第三方 Tushare 数据源怎么接？
A: 在 Tushare 数据源配置中填写 Token，并把第三方代理地址填到“自定义 HTTP URL”。保存后，系统级数据源中台会把该地址统一下发到所有 Tushare 调用链。
Q: QMT 行情通道需要新建单独模块吗？
A: 不需要。当前已经作为统一 `market_data` provider 接入，配置入口复用现有数据源中台；只要把 `source_type` 设为 `QMT (XtQuant)` 并补充本地参数即可。
Q: 配置后如何同步数据？
A: 目前需要手动在 Admin 后台执行数据同步脚本，或配置 Celery 定时任务自动同步。后续版本会提供一键同步按钮。
