## 六、冻结规则

在里程碑完成前，遵循以下冻结规则：

1. **暂停合入**：与本路线图无关的大改动
2. **允许合入**：
   - Bugfix（紧急修复）
   - 里程碑任务
3. **合入要求**：
   - 必须通过CI检查
   - 必须有对应测试
   - 必须更新相关文档
